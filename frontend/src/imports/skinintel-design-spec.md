Design a modern, minimal, fully responsive AI skincare web application called **SkinIntel**.

SkinIntel allows users to upload a photo or paste an image link of a skin concern (dark circles, acne, blackheads etc). AI analyzes the image and recommends skincare products available in the user's country within their selected price range.

The UI should follow a **modern AI + dermatology aesthetic**, similar to Apple Health, Notion AI, and premium skincare websites.

Design style requirements:

* Clean minimal interface
* Soft gradients
* Rounded components
* Glassmorphism cards
* Smooth shadows
* Mobile-first responsive design
* Professional healthcare aesthetic

Color palette:
Primary: #3EB6B1 (Soft Teal)
Secondary: #9C8CFF (Lavender)
Accent: #FFB6A3 (Peach)
Background: #F8FAFB
Text: #2B2B2B

Typography:
Headings: Poppins / Inter Bold
Body: Inter Regular
